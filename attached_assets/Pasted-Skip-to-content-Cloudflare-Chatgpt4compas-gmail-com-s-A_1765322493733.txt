Store relational data with serverless SQL database

D1 is Cloudflare’s native serverless database enabling you to build applications that handle large amounts of users at no extra cost.

type OrderRow = {
    Id: string;
    CustomerName: string;
    OrderDate: number;
};
export default {
  async fetch(request, env) {
    const result = await env.MY_DB.prepare(
        "SELECT Id, CustomerName, OrderDate FROM [Order] ORDER BY ShippedDate DESC LIMIT 100",
    ).run<OrderRow>();
    return new Response(JSON.stringify(result));
  }
}
